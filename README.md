# Nike Sales Dashboard

## Project Overview

This project analyzes Nike sales performance using interactive dashboards and data visualization techniques.

## Tools Used

* Tableau
* Python
* SQL
* Excel

## Key Insights

* Revenue and Profit Analysis
* Regional Sales Performance
* Customer Segment Analysis
* Product Category Performance
* Sales Trends and KPIs

## Files Included

* Tableau Dashboard (.twbx)
* Dataset (.csv)
* Dashboard Screenshot
* Project Report
